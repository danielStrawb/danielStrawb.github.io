TTY, or a good alternative to window managers

Recently i said my blog, that i want to try TTY, as alternative to window managers, anud ise Xorg only whem it's need. Soo... I show you some useful things. To use it, just press Alt+Ctrl+F(1/6)

It's like workplaces

You can use it like workplaces, and combine with tmux, to cmake effect of tiling window-managers, or just separate workplaces for fun, work, and something like thiss, like i do this in my WindowMaker.

Sometimes you don't need window manager

Programs, like mpv and links (in graphical mode), and others can run yourself in clear Xsession, without window manager, it's good, if you want a maximum minimalism, you don't need to run window manager, so your computer require less resources, there are list of software, that run yourself in clear Xorg, that i found:

  • mpv
  • links (to open a new tap, call "open new window" in context menu, and switch by tabs with Alt+(1-9), and close window with "q")
  • ffplay
  • OpenTTD
  • And others

    • As way to make minimal ecosystem

    With clear native tty, you can make a habit of using only applications, that give you only useful and important information, and run Xorg only when need to play videogame, or run applications, like XScreenSaver, or XSnow, to make winter on you monitor :D

    Enabling UTF-8

    To enable UTF-8 an TTY, type unicode_start in terminal, you can also automize this, by add this command in .bashrc

    What about other keyboard layouts?

    loadkeys, the only way to use another layouts in TTY, but it's require root, so you can do it manualy, by logining to root, and than execute loadkeys ru(as example), or add to /etc/rc.d/rc.local the same loadkeys ru, to have it done with startup.
